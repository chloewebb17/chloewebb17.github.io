Code Log 4: Color

CSS can interpret and apply color in a range of ways. Digitally, all colors have multiple different codes that express their exact appearance, such as HEX codes, RBG, and CYMK. CSS has a few different ways of implenting colors.

  1. Color

    2. This is the most basic way of applying color. When working with a specific element like text or a shape, color will do the trick.

  2. Background-Color

    3. This function only applies color to the very background of the page. Even when other elements aren't loading in, the background color you have chosen will show through.

  3. border-color

    4. This function only applies to the border around certain elements. Meaning, it will only show up if there's already a border in place. But that's true of every color, right? You can't color something that doesn't exist.
