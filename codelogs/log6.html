Code Log 6: Reference Quest

The element I chose to look at is the HTML head tag. At first, I thought the head was the header, but I now understand that it serves a purpose beyond the text you can see.

The sites I chose were W3Schools, Mozilla Developer, GeeksforGeeks, and Programiz.

All of these websites start out roughly the same, explaining the purpose of the head and showing an example in code, but W3schools immediately stands out because it is the only one that explains what metadata is. You see, the head contains metadata and other information regarding the webpage itself outside of the content of the site. Those new to programming might have no idea what metadata is (data about the document itself, not displayed within the document). Mozilla Developer and and Programiz go on to explain more complicated, higher level tags that can be found within the head, signifying that they are for more advanced developer resources, whereas W3schools assumes one is a beginner.
