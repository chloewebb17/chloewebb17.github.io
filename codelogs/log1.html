Code Log 1: HTML Semantic Tags

Semantic tags are HTML tags that clearly define their purpose to the browser and developer.

For example, a tag named "header" suggests that the text within the tag will be a header:

<header>I'm a header!</header>

Non-semantic vs Semantic tags

Not all tags are semantic. If you want to make text thicker, you could use either the semantic tag, or the non-semantic tag.

Strong actually indicates the function of the tag, but B is just a letter. As a designer, B seems a little more intuitive, but arguing about that would be... semantics (wink).
